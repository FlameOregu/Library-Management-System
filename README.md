# Library-Management-System
A Python library management system that uses object-oriented programming, inheritance, and encapsulation to manage books and user borrowing.

## Overview

This project is a library management system developed in Python using object-oriented programming principles. It allows users to borrow and return books while tracking availability and enforcing borrowing limits for different user types.

## Features

* Create and manage books.
* Track book availability.
* Borrow and return books.
* Standard and premium user accounts.
* Premium users have custom borrowing limits.
* Prevent borrowing of unavailable books.
* Display borrowed books for each user.

## Technologies Used

* Python
* Object-Oriented Programming (OOP)

## Concepts Demonstrated

* Classes and Objects
* Inheritance
* Encapsulation
* Method Overriding
* Lists
* Conditional Logic

## Future Improvements

* Add a book search feature.
